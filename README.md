## 📱 About The Project

**Redmi Smart Store** is a comprehensive **Inventory Management System** built in C that helps manage products, track orders, and handle customer purchases with a secure admin portal. Perfect for small to medium retail stores.

### ✨ Features

#### 🛍️ Customer Portal
- Browse products with real-time stock availability
- Search products by name or category
- Add/remove items to shopping cart
- Update cart quantities
- Place orders with delivery information
- Track order history and status
- Cancel pending orders

#### 👑 Admin Portal
- **Secure login** with Outlet ID + OTP verification
- Complete product management (Add, Update, Delete)
- Real-time inventory tracking
- Order management system
- Update order status (Pending → Processing → Shipped → Delivered)
- Cancel orders with automatic inventory restoration

#### 🔐 Security Features
- Two-factor authentication for admin access
- 6-digit OTP generation
- 3-attempt limit for failed logins
- Data persistence across sessions
